Данный сервис предоставляет возможность каждому учителю и студенту(или ученику) отвечать на созданные контрольные
в онлайне, а не на листочке. Если большинство листов, на которых делались работы выбрасываются, то в данном случае они
будут всегда храниться на сервере в удобном виде, всегда можно посмотреть что было и никогда не потеряется.
Что бы протестировать сервис необходимо:

зайти на: http://asiman.esy.es/
при регистрации указать группу: "demogroup" без кавычек
есть 1 студент : 
                login : "demologins" 
                password : "demopasswords"
есть 1 преподаватель : 
                login : "demologin"
                password : "demopassword"
что бы зайти в управляющую панель необходимо залогиниться с адресса : http://asiman.esy.es/teacher.html
везде к функционалу есть описание.
